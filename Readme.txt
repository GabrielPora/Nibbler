snake game:
-----------------------
-----------------------

	
to run type,

	./snake widthxheight

widthxheight = screen resolution of your monitor

ex:
	./snake 800x600


controles:
--------------
		
w,s,a,d - movement
p       - toggle pause
esc     - exit
